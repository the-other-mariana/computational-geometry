# Geometric Library v1.0

This is a first exercise where a custom library is to be done in order to perform further geometric operations. <br />

## Content

The `.py` file contains the following: <br />

- Points 2D
  - Constructor
  - Equals override
  - String override
  - rotation
- Line
  - Constructor
  - String override
  - Parallel
  - Intersection
  - Equivalence
- Static Functions
  - Distance (2 points)
  - Line (2 points)
